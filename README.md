# TrustProof

Cryptographic verification system for AI model outputs using lightweight proofs of inference.

## Features
- Merkle tree-based proof generation for AI outputs
- Zero-knowledge verification of model outputs
- Proof chain visualization and audit logs
- API for proof generation and verification
- Tamper-evident audit trails
- React + Tailwind SaaS webapp with Firebase Auth

## Quick Start
1. Clone this repo
2. Install: `pip install -e .`
3. Run: `trustproof --help`

## Web App
The `webapp/` directory contains a full SaaS frontend built with React + Tailwind CSS.
